#ProyectoAlice

Descripción del proyecto:
- El juego está enfocado en la experiencia del jugador, combinando exploración, movimiento avanzado y progresión mediante objetos.
A lo largo del nivel, el jugador desbloquea caminos y recibe información contextual para avanzar hasta la salida.

Mecánicas implementadas:
1) Sistema de Checkpoints: Guarda el progreso del jugador y permite retomar desde el último punto alcanzado.
2) Gancho de desplazamiento: Permite moverse rápidamente por el escenario y alcanzar zonas elevadas.
3) Wall Run: Mecánica de correr por las paredes para sortear obstáculos y acceder a nuevas áreas.
4) Puertas con objetos clave: Algunas puertas solo se abren al poseer un objeto específico.
5) Inventario simple: Gestión básica de objetos necesarios para avanzar.
6) Mensajes contextuales: Textos que se activan al llegar a determinados puntos del mapa para guiar al jugador.

Cómo ejecutar el proyecto:
 - Clonar el repositorio: git clone https://github.com/ChiaraB96/Alice.git
 - Abrir Unity Hub.
 - Seleccionar Open Project y elegir la carpeta del proyecto.
 - Abrir la escena principal y ejecutar el juego desde el editor.
